# Claude



- Project: iKong-server
- Created: 2026-04-09

## 프로젝트 개요

비접촉 mmWave 레이더 기반 독신 고령자 안전 모니터링 시스템
- 심장 박동 및 호흡 수 측정 기기 1개
- 낙상 감지 기기 1개
- 심작 박동 및 호흡 수 이상감지 시 앱을 통하여 보호자 한테 알림
- 낙상 감지시 앱을 통하여 보호자 한테 알림

## 기술 스택
- 백엔드: Spring Boot + JPA
- DB: ORACLE(메인), Supabase(개발할 때 사용, 배포 테스트 할 때 Oracle로 바꿀예정)
